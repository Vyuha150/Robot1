# Roadmap

## Phase 1: Working MVP

- Founder dashboard
- Employee task manager
- Lead manager
- Influencer database
- Student club database
- Product database
- Agent registry and orchestration
- Daily and weekly reports
- CSV import/export

## Phase 2: Execution Automation

- Morning and evening scheduled reports
- Employee scorecards
- Follow-up reminders
- Approval queues for outreach, legal drafts, political content, and mass communication
- Audit log review dashboard

## Phase 3: Advanced Modules

- Political intelligence and office operations
- Robotics tracking and pilot customer management
- Agri market intelligence
- Event planner
- Influencer agreement generator
- Opportunity monitor
- Vector memory

## Phase 4: Integrations

- Gmail
- Google Calendar
- Google Sheets
- WhatsApp Business API
- Notion or Airtable
- Approved social media APIs
